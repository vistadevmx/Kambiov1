Kambio POS - Fix Input Field Visibility

This tool will help fix the issue with invisible input fields in the sign-in form by creating a local script that applies custom CSS styles.

To use this fix:

  1. Click the "Apply Fix" button below
  2. Copy the generated code snippet
  3. In your browser's developer tools console (F12 or Command+Option+I), paste and run the code

Fix Script:

const injectStyles = () => {
  // Check if already injected
  if (document.getElementById('kambio-input-fix')) return;
  
  const styleEl = document.createElement('style');
  styleEl.id = 'kambio-input-fix';
  styleEl.textContent = `
    /* Fix for input fields */
    input[type="text"],
    input[type="password"],
    input[type="email"],
    input {
      background-color: rgba(255, 255, 255, 0.95) !important;
      border: 1px solid rgba(203, 213, 225, 0.8) !important;
      color: #1e293b !important;
      min-height: 48px !important;
      font-size: 16px !important;
      display: block !important;
      width: 100% !important;
      padding: 12px !important;
      border-radius: 12px !important;
      margin-bottom: 12px !important;
      opacity: 1 !important;
    }
    
    /* Updated card style */
    div[data-style] {
      border: 1px solid rgba(203, 213, 225, 0.8);
    }
  `;
  
  document.head.appendChild(styleEl);
  console.log('Kambio POS: Input fix applied successfully!');
};

// Run immediately and also set up to run when DOM content loads
injectStyles();
document.addEventListener('DOMContentLoaded', injectStyles);

// Also run periodically in case of dynamic content
setInterval(injectStyles, 2000);
